# To-Do List App

A simple, modern, and responsive To-Do List web application built with React + TypeScript + Vite + Tailwind + Shadcn UI.

It helps users manage daily tasks efficiently with a clean UI and persistent storage.

## Live Demo

[View Live Demo](https://to-do-list-app-yjj4.vercel.app/)

## Features

- Add new tasks with title and details
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Data persistence using Local Storage
- Task summary (Completed / Pending / Total)
- Built-in calendar view
- Clean and responsive UI

## Tech Stack

- **Frontend**: React, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Custom + shadcn/ui
- **Icons**: React Icons
- **Deployment**: Vercel

## Installation & Setup

Clone the repository:

git clone https://github.com/Sukhdeep-Singh-Panesar/To-Do-List-App

### Navigate into the project directory:

cd to-do-list-app

Install dependencies:

npm install

### Run the development server:

npm run dev

### Build for production:

npm run build

Preview the production build:

npm run preview
